Upgrading from version 1.4

If you have played KulkiEx in version 1.4 before and you want to have all your saved games and highscores you have to make only two simple operations. Assuming that you have installed new version in "C:\KulkiEx2\", and the old version is in "C:\KulkiEx":

     Copy file "C:\KulkiEx\wyniki.dat" to folder "C:\KulkiEx2\".

     Copy files with *.kex extension (they are kept in "zapisy" folder by default) to "C:\KulkiEx2\save".

Now you have your old highscores and saved games in version 2.0 :-)
