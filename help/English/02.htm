Short overview

KulkiEx is a logic game. During the game you move the balls in that way they will create a line. If the line is made from at least five balls it is removed from game field. In case when movement does not end with strike there are added three balls in random colors and positions. Your goal is to try to create lines in each movement. For each striked ball you get suitable number of points. The games ends when you cannot make any move.

Movement
The ball moves only in vertical and horizontal lines.

    Choose with the mouse ball which you want to move.
    Choose with the mouse an empty field on which the ball should move.
    The ball will move only if there is a passage to the destination.

Strikes
Strike is made when on the game filed there are at least five balls which are creating a line. All balls creating a line are removed from game field. In case when there are more than a one line all the lines are removed.
