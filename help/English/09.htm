Settings

After choosing "Settings" from menu you'll see window in which you can change game options:

     do you want to play with or without sounds

     do you want to see next balls, which will be added afater no striked movement. This option has influence on points.

     youn can choose on of three possible settings: 5, 7, 9 colors, each option has its own number of points for strike. The higher number of colors the higher is game difficulty.

     you can choose language in which the game fill communicate with you ;-) You can read more in "Translations" section.

The green button is saving the settings the red one closes the window without saving them.
